凯丽公式

f* = (bp - q) / b = (p(b+1) - 1) / b

其中：

f* = 应该投入的资金比例

b = 盈亏比 (盈利金额 / 亏损金额)

p = 盈利概率 (此处假设为50%)

q = 亏损概率 (1 - p)

凯丽公式解释

凯丽公式帮助投资者确定在每次投资中应投入多少资金，以最大化长期回报同时避免破产风险。

风险提示

凯丽公式提供理论上的最优仓位，实际投资中应考虑个人风险承受能力、市场波动性等因素。计算结果仅供参考，不构成投资建议。
